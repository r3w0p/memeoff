Meme battling with friends.


## Setup

1) Install packages using `npm install`
2) Run `npm start`, or `node index.js` to specify parameters (below)
3) Go to `http://localhost:<port>`


## Parameters

- `--port`, the port (*default*: `3000`)
- `--submit`, duration in which players can make submissions, in seconds (*default*: `60`)
- `--vote`, duration in which players can vote, in seconds (*default*: `30`)
- `--winner`, duration to show the winning meme, in seconds (*default*: `15`)
